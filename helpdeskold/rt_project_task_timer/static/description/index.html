⏱️ Temporizador de Tareas de Proyecto

Seguimiento Automático de Tiempo con Alertas Visuales

Rastrea el Tiempo Automáticamente en Cada Tarea

Mantén tus tareas de proyecto bajo control con seguimiento automático de tiempo que inicia en el momento que creas una tarea y proporciona retroalimentación visual en tiempo real basada en el tiempo transcurrido.

Características Principales

🚀

Inicio Automático del Temporizador

El temporizador inicia automáticamente cuando creas una nueva tarea. ¡No se requiere intervención manual!

⏸️

Detención Automática Inteligente

El temporizador se detiene automáticamente cuando mueves la tarea a una etapa de conclusión (etapas plegadas).

🎨

Alertas Codificadas por Color

Indicadores visuales de color en vistas kanban y formulario basados en umbrales de tiempo transcurrido.

⚙️

Umbrales Configurables

Establece tus propios umbrales de tiempo para alertas amarillas y rojas en Configuración.

🌍

Seguimiento de Tiempo en UTC

Seguimiento de tiempo consistente para todos los usuarios sin importar su zona horaria.

📊

Visualización en Kanban

Visualización del temporizador en tiempo real en vista kanban con badges codificados por color (formato HH:MM:SS).

Alertas Visuales de Tiempo

Las tareas se codifican automáticamente por colores según el tiempo transcurrido para ayudarte a identificar tareas que necesitan atención.

🟢 Verde

Menos de 1 hora

(Por defecto)

🟡 Amarillo

1-2 horas

(Advertencia)

🔴 Rojo

Más de 2 horas

(Alerta)

💡 Nota: Los umbrales de tiempo son totalmente configurables en Configuración → Proyecto → Configuración del Temporizador

¿Por Qué Usar el Temporizador de Tareas?

  • Mejora la Gestión del Tiempo: Ve instantáneamente qué tareas están tomando demasiado tiempo
  • Aumenta la Productividad: Las alertas visuales ayudan a los miembros del equipo a mantenerse enfocados
  • Mejor Planificación: Usa los datos de tiempo acumulado para mejorar estimaciones futuras
  • Seguimiento Automático: No necesitas recordar iniciar/detener temporizadores manualmente
  • Consistencia del Equipo: El seguimiento basado en UTC asegura que todos vean el mismo tiempo
  • Configuración Sencilla: Ajusta los umbrales de tiempo para que coincidan con el flujo de trabajo de tu equipo
  • Historial Completo: Rastrea fecha de inicio, fecha de fin y tiempo total acumulado
  • Integración con Kanban: Ve el estado del temporizador directamente en tu tablero kanban

Instalación

1

Descarga el Módulo

Descarga el módulo rt_project_task_timer desde la tienda de Odoo Apps

2

Instala en Odoo

Ve a Aplicaciones → Cargar Módulo → Selecciona el archivo ZIP → Instalar

3

Configura (Opcional)

Navega a Configuración → Proyecto → Configuración del Temporizador para ajustar los umbrales de tiempo

4

Comienza a Usar

¡Crea una nueva tarea y el temporizador iniciará automáticamente!

Configuración

Configuración de Umbrales de Tiempo

Navega a Configuración → Proyecto → Configuración del Temporizador para configurar:

  • Horas para Alerta Amarilla: Establece el umbral para advertencia amarilla (por defecto: 1 hora)
  • Horas para Alerta Roja: Establece el umbral para alerta roja (por defecto: 2 horas)

Cómo Funciona

El temporizador sigue estas reglas:

  • El temporizador inicia automáticamente cuando creas una nueva tarea
  • El temporizador continúa corriendo mientras la tarea está en etapas activas
  • El temporizador se detiene automáticamente cuando la tarea se mueve a una etapa plegada/concluida
  • El temporizador se reinicia automáticamente si mueves la tarea de vuelta a una etapa activa
  • Todos los tiempos se almacenan en UTC para consistencia entre zonas horarias
